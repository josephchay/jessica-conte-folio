## Development

Clone the repository:

```bash
git clone https://github.com/josephchay/jessica-conte-folio
```

Navigate to the project directory:

```bash
cd jessica-conte-folio
```

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
